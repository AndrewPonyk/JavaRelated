Application.show

id: ${rederArgsId} - get from params HashMap
id: ${id} - get from action arguments , if we pass wrong value , Play cant cast it , so it will be null
If we make 'routes' :
GET /showParams/{id} Application.showParams

make action in controller
public static void showParams(Long id ){..}

and make request http://localhost:9000/showParams/10
we will have 'id' in 'params' hashmap , and in method argument

We can pass multiple values for a single URL parameter

http://localhost:9000/showParams/10?id=100&id=1000

in 'routes'
GET /showParams Application.showParams

in action in Controller public static void showParams(List<Long> id){...}

and on page we can get $ {id.get(0)} - it will be 10
$ {id.get(1)} - it will be 100
$ {id.get(2)} - it will be 1000
