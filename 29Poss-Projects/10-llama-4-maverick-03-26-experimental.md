## Notes on IDE Selection

**By Technology:**
- **JVM Languages (Java, Kotlin, Scala):** IntelliJ IDEA Ultimate (commercial) or Community (free), Eclipse
- **Python:** PyCharm Professional (commercial) or Community (free), VS Code with Python extension, Jupyter Lab/Notebook for data science
- **.NET/C#:** Visual Studio (Windows), JetBrains Rider (cross-platform), VS Code with C# extension
- **Go:** GoLand (commercial), VS Code with Go extension
- **Rust:** RustRover (commercial), VS Code with rust-analyzer
- **C/C++:** CLion (commercial), Visual Studio (Windows), VS Code with C++ extension, Qt Creator (for Qt projects)
- **PHP:** PHPStorm (commercial), VS Code with PHP extensions
- **Ruby:** RubyMine (commercial), VS Code with Ruby extension
- **JavaScript/TypeScript:** VS Code, WebStorm (commercial), Sublime Text
- **Flutter/Dart:** Android Studio with Flutter plugin, VS Code with Flutter extension
- **Game Development:** Unreal Engine Editor, Unity Editor with Visual Studio, Godot Editor
- **Database Work:** DBeaver (universal), DataGrip (commercial), pgAdmin (PostgreSQL), MySQL Workbench, MongoDB Compass, Neo4j Desktop
- **Web Design:** Figma, Adobe XD, Sketch (macOS only)
- **Data Science/ML:** Jupyter Lab, PyCharm, RStudio (R), Databricks Notebooks
- **DevOps/Infrastructure:** VS Code, PyCharm, terminal-based (vim, emacs)

**General Purpose Editors:**
- **VS Code:** Extremely versatile with extensions for almost every language
- **Sublime Text:** Lightweight, fast, good for quick edits
- **Vim/Neovim:** Terminal-based, highly customizable, steep learning curve

**Specialized IDEs:**
- **Android Studio:** Android app development (Flutter, Kotlin, Java)
- **Xcode:** iOS/macOS development (required for Apple platforms)
- **STM32CubeIDE/Keil µVision:** Embedded systems development
- **Remix IDE:** Solidity/blockchain smart contracts (web-based)
