3. What is the difference between raster and vector data in GIS?

Answer: Vector data uses points, lines, polygons, precise boundaries, smaller file size, scalable, suitable for discrete features (roads, buildings). Raster data uses grid cells/pixels, continuous data, larger files, resolution-dependent, suitable for continuous phenomena (elevation, temperature). Choose based on data type and analysis needs.

6. How do you implement spatial indexing in PostGIS?

Answer: Create spatial index with `CREATE INDEX idx_geom ON table USING GIST(geom)`. GIST (Generalized Search Tree) enables fast spatial queries. Use `ANALYZE` to update statistics, `VACUUM` to optimize. Spatial indexes dramatically improve query performance for large datasets.

9. What is the difference between geographic and projected coordinate systems?

Answer: Geographic coordinates use latitude/longitude on ellipsoid, angular units (degrees), global coverage, distorted distances/areas. Projected coordinates use x/y on flat plane, linear units (meters), regional coverage, preserves specific properties (area, distance, shape). Project geographic data for accurate measurements.

15. What is the difference between Shapefile and GeoPackage?

Answer: Shapefile is older format, multiple files (.shp, .shx, .dbf), 2GB limit, limited data types, no topology. GeoPackage is modern SQLite-based, single file, no size limit, rich data types, supports raster/vector, topology, better performance. GeoPackage is recommended for new projects.
