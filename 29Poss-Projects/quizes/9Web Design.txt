15. What is the difference between CSS Grid and Flexbox?

Answer: CSS Grid is 2D layout system (rows and columns), best for page layouts, can overlap items. Flexbox is 1D layout system (row or column), best for component layouts, items don't overlap. Grid for macro layouts, Flexbox for micro layouts. Grid has explicit placement, Flexbox has content-based sizing.

18. How do you create animations in CSS?

Answer: Use `@keyframes` to define animation steps, then apply with `animation` property. Specify duration, timing function, delay, iteration count, direction. Example: `@keyframes slide { from { left: 0; } to { left: 100px; } }` then `.element { animation: slide 2s ease-in-out; }`. Also use `transition` for simple property changes.

21. What is the difference between rem and em units?

Answer: `rem` (root em) is relative to root element's font size, consistent across nested elements. `em` is relative to parent element's font size, compounds with nesting. Use `rem` for consistent sizing, `em` for proportional scaling. 1rem = root font size (usually 16px), 1em = parent font size.
