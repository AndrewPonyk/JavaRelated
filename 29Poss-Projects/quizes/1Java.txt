4. Explain the difference between `@Transactional(propagation = Propagation.REQUIRED)` and `@Transactional(propagation = Propagation.REQUIRES_NEW)`.

Answer: REQUIRED joins existing transaction or creates new one if none exists. REQUIRES_NEW always creates a new transaction, suspending the current one if it exists. REQUIRES_NEW is useful for logging or audit operations that should commit regardless of main transaction outcome.

10. Explain the difference between JPA's `EntityManager.persist()` and `EntityManager.merge()`.

Answer: persist() makes transient entity persistent, throws exception if entity already exists. merge() copies state of detached entity to persistent entity, returns managed entity. Use persist() for new entities, merge() for updating detached entities.
